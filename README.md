# Simple Java Calculator
This is a **basic calculator** built in java using 'Scanner' and 'if-else-if' conditions .

## Features
supports '+','-','*', and '/'
Takes user input for two numbers and operators
Handles division by zero with an error message

## How to run
1.compile the file
2.run the program
